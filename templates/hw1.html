언어인지과학과 202101601 박해준

Problem Statement Canvas

고객

  • 시간표를 짜는 한국외대 학생들

컨텍스트

  • 강의실이 위치할 건물을 알 수 없어 연강을 계획할 때 애로사항 발생

근본문제

  • 강의실이 위치가 수강신청 전에 제공되지 않아 합리적인 의사결정을 하지 못함

감정적 영향

  • 불안
  • 답답함
  • 시간표 확정에 대한 불확실성

정량적 영향

  • 먼 강의실의 강의끼리 연강 시 지각하기 쉬움
  • 시간표 재조정 시 부담 증가

대안

  • 직접 과거 강의의 강의실 검색 후 예측
  • 전공에 따른 강의실 편향으로 예측

대안의 단점

  • 과거의 강의실 배정과 같다는 보장 없음
  • 신설 강의의 경우 예측 어려움

솔루션

  • 과거 강의실 배정 기반 강의실 건물 예측
  • 시간표 작성 시 연강 가능 여부 분석
  • 강의 건물과 캠퍼스 지도 연동

기능 명세

  • 강의명, 교수명에 따른 예상 강의실 건물 위치 제공
  • 신설 강의 혹은 데이터 없는 강의의 경우 전공, 교양 등 강의 성격 바탕으로 예상 위치 제공
  • 시간표 작성 시 연강 여부 및 연강 시 예상 건물 표시
  • 당일 예상 경로를 지도에 표시

간단한 UX 흐름

  • 사용자 해당 학기 입력
  • 강의 선택 및 시간표 작성
  • 강의명과 함께 강의실 예상 위치 제공
  • 연강 검토
  • 지도에 당일 경로 표시
